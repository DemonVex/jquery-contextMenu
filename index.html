jQuery contextMenu by arnklint

Simple, small, yet powerfull and easy to use context menu when right clicking on something for jQuery. Documentation

Example 1

Lets look at a single div as a target

Note that the native context menu is disabled by passing {disable_native_context_menu: true} as the options hash and last argument of the plugin. The native context menu is enabled by default.
